# Facebook Marketplace Ad Publishing Bot

A marketplace bot which will publish ad automatically.

## How to Use
1. Install all the dependencies using this command. Skip if you already have the required packages.
<br>`pip install -r requirements`
2. Create the `ads.csv` file containing ad information.
3. For devices condition and device model codes (to be used in `ads.csv` file) refer to `Device Models.html` file.
4. Run the script simply using `python bot.py`
5.Download google chromoe driver. ( https://chromedriver.chromium.org/downloads )
6.Your system should have python installed ( https://www.python.org/downloads/ )
7. Keep all files in one folder.
